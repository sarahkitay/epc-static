# Elite Performance Clinic - Static Site

## Client Management System

A password-protected internal client management system for Elite Performance Clinic.

### Quick Start

#### Option 1: Using npm/npx (Recommended)
```bash
npm run dev
```
or
```bash
npx serve . -l 3000
```

#### Option 2: Using the dev script
```bash
./dev.sh
```

#### Option 3: Using Python
```bash
python3 -m http.server 3000
```

### Access the System

Once the server is running:
- **URL:** http://localhost:3000/clients/
- **Password:** `15225`

### Deploy to Vercel

The site is ready for Vercel deployment:
```bash
vercel
```

Or connect your GitHub repo directly in the Vercel dashboard.
